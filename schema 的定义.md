# schema 的定义

## 首屏

背景：支持图片，默认用主题色兜底

badge：emoji + text

主标题：text

副标题：text

CTA按钮：

副按钮：

背书文字：text

产品展示：img or video

## 数据展示模块

主标题：text

副标题：text

数据项：多个数据项，每个数据项（bgImg + icon + text + label）

## 套餐模块

主标题：text

副标题：text

套餐项：支持多个，每个套餐项(name + desc + badge + label + 价值点(多行文本) + 倒计时 + cta 按钮)

## 产品模块

主标题：text

副标题：text

产品展示项：支持多个，每个展示项（name + desc + bgImg）

## 前后对比模块

主标题：text

副标题：text

免责声明：text

对比项：支持多个（crmName + 使用时长 + 案例描述 + afterImg + beforeImg）

## 服务流程模块

主标题：text

副标题：text

步骤项：支持多个（步骤标题 + 步骤描述 + img）

## 信任模块

背景图：img

信任徽章：支持多个（icon + 主文案 + 副文案）

## 特性模块

主标题：text

副标题：text

特性项：支持多个（icon + tile + desc）

## 评价模块

主标题：text

副标题：text

desc：text

评价项：支持多个（avater + name + location + channel + 评价内容（text + img））

## 产品故事

主标题：text

副标题：text

正文段落：desc

背景：img

署名：text

职位：text

## 倒计时模块

主标题：icon + text

副标题：text

截止时间：date

## 常见问题模块

主标题：icon + text

副标题：text

问题项：支持多个（q + a）

## 安全保障模块

主标题：text

副标题：text

desc：text

保障项：支持多个（icon + 主标题 + 副标题）

## 页脚模块

品牌名称：text

版权年份：text

联系邮箱：email

隐私政策：text

服务条款：text

## 悬浮按钮

按钮文案：text

按钮链接：link
